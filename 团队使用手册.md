# Sino Gear CRM 团队使用手册

> 写给销售团队的日常使用说明。技术细节看 `CLAUDE.md`，这里只讲怎么用。

---

## 一、这是什么

一个装在 Chrome 上的扩展，打开 WhatsApp Web 时会自动叠在上面。功能：

- 客户资料自动归档（从聊天里提）
- 实时翻译客户消息
- AI 写回复（接你自建的 Gemini Gem）
- 车源库 + 一键发图给客户
- 任务/报价/标签/看板

**所有数据存在云端 (Supabase)，团队成员共享同一份客户库。但每个销售默认只看自己跟的客户**（顶栏可切换"全部"看团队全貌）。

## 二、第一次安装（每个人都要做）

### 1. 装 Chrome 扩展

1. 找经理要 `dist.zip`（或者扩展商店地址，如果上架了）
2. 解压到一个固定文件夹（**别删，扩展从这里加载**）
3. Chrome 地址栏输入 `chrome://extensions/`
4. 右上角打开 **"开发者模式"**
5. 左上角 **"加载已解压的扩展程序"** → 选刚才的文件夹
6. 看到一张绿色图标的卡片就成功了

### 2. 登录 Chrome 自己的 Google 账号

扩展会用 Chrome 当前登录的 Google 账号同步联系人 + 跑 Gem。**用公司号登录，别用私人号。**

### 3. 注册 CRM 账号

1. 点浏览器右上角扩展图标 → 弹窗里填邮箱密码 → 注册
2. 注册后**先不要建团队**，让经理把你拉进去（见下一节）

### 4. 打开 WhatsApp Web

去 https://web.whatsapp.com 用工作号扫码登录。登录后顶部会出现绿色导航栏（看板/聊天/客户/车源/任务/标签），就是扩展加载好了。

## 三、加入团队（管理员操作）

**新员工注册后，告诉经理你的邮箱。**

经理这边：
1. 顶栏点 **👥 团队** → "邀请成员"
2. 填新员工邮箱 + 角色（一般选 `member`）
3. 新员工那边刷新 WhatsApp Web，就能看到全部客户数据了

角色说明：
- `owner` — 老板，啥都能干
- `admin` — 主管，能管成员
- `member` — 普通销售，能改客户但不能动团队成员

## 三·五、视图范围 + 撞单提醒（重要！）

### 顶栏 `👤 只看我的 / 🏢 全部` 切换

顶栏 logo 旁有个**绿色胶囊下拉**，俩选项：

| 选项 | 含义 | 默认 |
|---|---|---|
| 👤 **只看我的** | 只显示你跟过的客户（聊过 / 打开过 / 你建的） | 销售默认 |
| 🏢 **全部** | 团队所有客户（适合经理 / 接同事单时） | 经理默认 |

**影响范围：** 看板 KPI / 聊天 tab 客户列表 / 客户 tab / 任务 tab / 跟进提醒。**车源 / 标签 / Gem 模板** 不受影响。

**搜索是例外**：客户 tab 用搜索框时永远搜全部库（方便查同事的客户），不受 scope 限制。

### 撞单提醒 ⚠️

如果同一个客户有 ≥ 2 个销售跟过，客户名字旁会出现橙色 **`撞单：张三`** tag。

**触发条件：** 双方都在 WhatsApp 里打开过这个聊天 → CRM 自动登记两人都是 handler → 出现撞单 tag。

**看到撞单怎么办：**
1. 内部沟通谁主跟（避免给客户报两个不同价）
2. 退出方关掉 WA 那个聊天 + 不再打开（不会自动取消归属，但当前不再"撞")
3. 经理可以直接在 Supabase 删掉错误的 contact_handlers 行

### 我的客户怎么"获得"

打开 CRM 第一次会自动把你 WhatsApp 里现有的所有聊天登记到你名下。之后每次：

- **进 WhatsApp 某个聊天** → 自动登记你为 handler（心跳）
- **手动建客户 / 跑 "同步 WhatsApp 聊天"** → 创建者自动是 handler

如果你想接管同事客户：切到 "🏢 全部" → 找到那个客户 → 在 WhatsApp 打开聊天一次 → 你也成为 handler（同时出现撞单 tag）。

## 四、顶部 6 个 Tab 怎么用

### 看板 📊
首页，看本周/本月业绩：
- 6 个 KPI 卡（新客户/报价/成交...）
- 阶段漏斗
- 热门车型 Top 5

每天上班先看一眼。

### 聊天 💬（最重要）

打开 WhatsApp Web 对话时自动激活。

**布局：**
- **左边**：筛选条件（按阶段/质量/区域/车型/预算/今日待办）
- **中间**：筛选出来的聊天列表
- **右边**：当前客户的资料卡，3 个 tab：
  - **客户资料** — 国家/语言/预算/标签/车型/报价/任务/时间线
  - **AI 回复** — Gem 写的回复 + 推荐车源
  - **历史消息** — 翻聊天记录

**填客户资料：** 大部分字段 AI 会自动抽（看下面 AI 章节），手动改也行，改完自动存。

### 客户 👤
全部客户列表 + 搜索 + 阶段筛选。每行有个 **💬** 按钮，点了直接跳到 WhatsApp 该客户的聊天。

### 车源 🚗
公司车源库。
- **新建/编辑车型**：填品牌/年份/价格/阶梯价/媒体（图片/视频/配置表）
- **媒体管理**：拖文件进去自动传 Cloudinary
- 客户聊天 → AI 回复 tab 上方会自动推荐匹配车型

### 任务 ✅
- 顶部 4 个 KPI（今日/本周/累计待跟进/总数）
- 日历：每天写有任务的客户名 + "+N" 表示溢出
- 点某天看详情列表

任务可以手动建，也可以让 AI 在客户卡里建议（见下面）。

### 标签 🏷
全公司标签管理：改名/合并/删除。**改名是全局的**，会影响所有客户，慎用。

## 五、AI 功能怎么用

### 1. 自动翻译聊天 🌐

顶栏右边有 **🌐 翻译** 开关。打开后客户发的外语消息会在原文下面显示中文。

- **🔁 重译** 按钮：翻译错了点这个重新翻
- **每条消息悬停** → 出现 🌐 按钮，可以单条手动翻
- 中文消息自动跳过（不会翻自己说的）

### 2. AI 自动抽客户资料

**打开聊天 → 等几秒** 资料卡里的国家/语言/预算/目的港/车型兴趣会自动填进去。

- 如果没抽到 → 聊天还不够长，等客户多说几句
- 抽错了 → 直接手动改，改完会覆盖 AI 的值
- 如果客户给了多个车型 → 都抽进 vehicle_interests

### 3. AI 标签建议 / 任务建议

- **标签 section** → 🤖 AI 建议 → 给销售特征标签（比如 "T/T 付款""急单""比价中""有抗性"）
- **任务 section** → 🤖 AI 建议 → 给下一步动作（比如 "明天发 Hilux 报价" + 截止日期）

点一下采纳，不喜欢就忽略。

### 4. Gem AI 回复（重磅）

**Gem 模板是每人各自一份**——你 Google 账号下建的 Gem，别人登录别的 Google 号打不开你的链接。所以**每个销售要自己在 https://gemini.google.com/gems 建 Gem，然后在扩展里录入。**

详细步骤见本文档最后的"Gem 配置流程"章节。

操作：
1. 客户卡 → **AI 回复** tab → 顶部下拉选 "Gem"
2. 选模板（默认那个会自动选 — **是你自己的默认**，不是团队共享的）
3. **前台开关**：
   - 关 = 后台跑（看不到 Gemini 标签页，推荐）
   - 开 = 前台跑（调试时看 Gem 在干嘛）
4. 点 **生成回复** → 等 1-3 分钟（Pro 模型慢但聪明）
5. 出来三段卡片：
   - **WhatsApp Reply** — 给客户的话（中/英/客户语言看 Gem 设置）
   - **Translation & Strategy** — 翻译 + 销售策略说明
   - **Client Record** — Gem 抓到的客户信息

**怎么用：**
- 看 reply 满意 → 点 **💬 填入聊天框** → 自动 paste 到 WhatsApp 输入框（**不会自动发送**，自己审一遍按发送）
- 不满意 → 下面有续聊框，写"再短一点""加上 CIF 价格"之类的，Cmd/Ctrl+Enter 发送，Gem 重写
- Client Record 想存 → 看跟现有资料的 diff → 点 **应用 N 项到客户资料**

**指令草稿：** Reply 上方有个常驻 textarea，写"重点强调付款周期"之类的固定指令，扩展会自动注入到 prompt 顶部。**每个客户单独存**，下次切回来还在。

### 5. 车源推荐

AI 回复 tab 上方有"相关车源"卡片，根据客户的 vehicle_interests 自动从车源库匹配。

- 卡片上有图册 / 视频 / 配置表（PDF）
- 点 **💬 一键发图** → 直接把图片/视频/PDF 塞到 WhatsApp 输入框，按发送就发出去了

比手动翻文件夹找图快很多。

## 六、抓客户发的图/视频/资料到车源库

客户经常会发别家车的图问"这个有没有"，或者发自己的需求资料。可以一键存到我们车源库。

**单张图/视频：**
- 鼠标悬停消息上的图 → 右上角出现 📥 按钮 → 点

**多张/相册/PDF/Excel：**
1. 在 WhatsApp 长按一条选中（进入多选模式）
2. 选要保存的所有消息（图/视频/文档）
3. 顶部出现 **📥 加入车源** 按钮 → 点

**保存后：**
- 屏幕右下角弹出"暂存盘"小窗 → 显示已捕获的缩略图
- 点 **📤 保存到车型** → 选已有车型 / 新建车型（填品牌+型号即可）→ 上传

**⚠️ 暂存盘刷新页面就清空**，赶紧分配。

## 七、客户分级 + 跟进提醒

### 质量分级（资料卡顶部）
- ⭐⭐⭐ **大客户** — VIP 待遇，优先回
- ⭐⭐ **有潜力** — 默认值
- ⭐ **普通**
- 🗑 **垃圾** — 骗子/同行/学生

筛选时可以按这个过滤。

### 阶段（customer_stage）
顺序：`新客户 → 资格审查 → 谈判中 → 暂时搁置 → 已报价 → 成交 → 流失`

- 大部分阶段会**根据聊天活跃度自动写回**（比如多日没回 → stalled）
- `已报价 / 成交` 是粘性的，不会被自动改
- 自己也可以手动改

### 跟进提醒
客户进入 `stalled`（搁置）时左侧"今日待办"会出现"待跟进"。处理完点确认，不想再提醒就关掉。

## 九·五、Gem 配置流程（每个销售第一次都要做）

每人 Gemini 账号不通用，**Gem 模板是 per-user 的**——你建的别人看不到，别人建的你看不到。每个销售自己跟一遍：

### 步骤 1：建 Gem
1. 登录 https://gemini.google.com/gems （用你的工作 Google 账号）
2. 点 **新建 Gem**
3. 写 system prompt（找经理要模板文案，建议输出 `[Client Record]` / `[WhatsApp Reply]` / `[Translation]` 三段以便扩展解析；不写也行，会按 CJK 比例自动拆分）
4. 选 **Pro 模型**（慢但聪明，适合销售）
5. 保存
6. **复制这个 Gem 的 URL**（形如 `https://gemini.google.com/gem/xxxxx`）

### 步骤 2：CRM 扩展里录入
1. WA Web 上扩展顶栏点 **🤖 Gem**
2. 点 **+ 新建模板**
3. 粘贴刚才的 Gem URL + 起个名字
4. 勾 **设为默认模板** → 创建

### 步骤 3：试用
1. 打开任意客户 → AI 回复 tab → 选 Gem
2. 第一次跑：勾 **前台开关**（看 Gemini 自动操作的过程）
3. 点 **生成回复** → 等 1-3 分钟
4. 工作正常 → 后续把前台关掉，改后台跑

### 报错 GEMINI_AUTH_REQUIRED 怎么办
浏览器另开一个 tab 去 https://gemini.google.com 手动登录一次（必须是录 Gem URL 那个 Google 账号），然后回 CRM 重新生成。

## 十、常见问题

### 装好了但 WhatsApp Web 上没有顶部导航栏？
- 刷新 WhatsApp Web
- 还不行 → `chrome://extensions/` 看扩展有没有报错红字 → 截图给经理

### 翻译/AI 抽取没反应？
- 没网 / API 限流 → 等 1 分钟再试
- 一直不行 → 找经理看 Qwen 额度

### Gem 生成卡死 / 报错 GEMINI_AUTH_REQUIRED？
- 浏览器另开 https://gemini.google.com 手动登录一次（用公司 Google 号）
- 再回来重新生成

### 改了客户资料发现回到旧值？
- 多人同时编辑同一客户 + 自动同步会冲突
- 改完等几秒再切走，让它存到云

### 客户找不到？
- **客户 tab** 搜索手机号或名字
- WhatsApp 搜不到 → 先在 WhatsApp 通讯录加这个号

### 切换电脑还能看到客户吗？
能。所有数据在云端，新电脑装好扩展登同一个邮箱就行。WhatsApp 那边是另外的扫码登录，跟 CRM 无关。

### 误删了客户/标签怎么办？
找经理 / 数据库管理员从备份恢复（pg_cron 心跳保证 Supabase 不会自动暂停）。**自己别乱删，没有撤销。**

### 我"只看我的"是 0，怎么办？
- 还没在 WhatsApp 里打开过任何聊天 → 随便点一个客户的聊天进去，会自动登记
- 或者顶栏左侧"维护工具" → 点 **同步 WhatsApp 聊天** → 一次性把 WA 现有聊天导入并归到你名下

### 我看到一个客户名字旁有"撞单：xxx"
表示同事 xxx 也在跟这个客户。立刻去找他对一下：
- 谁主跟，避免给客户报两个不同价
- 不主跟的人不要再发消息（CRM 不会自动取消归属，但实际行动上谁不发就算谁退出）

### 我同事建的 Gem 我能用吗？
**不能。** Gem URL 只对建它的 Google 账号有效，每个销售必须自己建。但 prompt 文案可以共享 — 找经理要文案，复制粘贴到自己 Gem 里。

### 我接同事走了的客户怎么办？
1. 顶栏切到 **🏢 全部**
2. 客户 tab 搜手机号 / 名字
3. 点 💬 跳到 WhatsApp 那个聊天 → 你也成为 handler
4. （可选）让经理在 Supabase 删掉前同事的 contact_handlers 行，撞单 tag 就消失了
